Campus Nexus

Campus Nexus is a full stack web application that connects college seniors and juniors. Seniors can upload items, and juniors can view and select them.

Features

Seniors can add items with name, price, and image
Juniors can browse items and view details
Junior can select an item and get senior contact details
Selected items are removed so others cannot see them
User login and signup with role based access

Technologies

Frontend HTML CSS JavaScript
Backend Node.js and Express
Database SQL Server
Multer for image upload

How it works

User signs up and logs in
Senior uploads item which is stored in database
Junior views items from database
Junior selects item and gets contact details
Item is removed after selection
